# Lego-Inventor-Manager

### Lego-Inventor-Manager est une application web permettant de gérer les pièces du kit Lego Mindstorms 51515 et de consulter les montages disponibles et d’administrer les caisses et pièces grace à une interface pour admin.

### Projet réaliser par : Mateo Grgic, André De Carvalho Varela Batista, Léonard Weber et Edis Ahmetaj
